# Ecommerce Sales Dashboard – Power BI Project

## Objective
Analyze ecommerce sales performance using interactive Power BI dashboards.

## Key Metrics
- Total Sales
- Orders & Revenue
- Top Products
- Regional Performance

## Visuals Used
- KPI Cards
- Bar & Column Charts
- Line Chart – Sales Trend
- Tables with Conditional Formatting
- Ribbon chart
- Tree map
- Area Chart
- Maps
- Waterfall Chart
- Donut Chart
- Slicers

## Tools
- Power BI
- DAX
